# Graph Homomorphism Notes

Minimal repository for notes and presentation material on graph homomorphism bounds.

## Structure

- `paper/` contains the source paper PDF.
- `notes/` contains study notes and related Markdown/TeX notes.
- `slides/` contains presentation sources.
- `drafts/` contains older draft TeX files.

## Build

Generate the Beamer slides from the repository root with:

```bash
cd slides
pdflatex presentation_beamer.tex
pdflatex presentation_beamer.tex
```
